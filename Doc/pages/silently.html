The <<silently>> Macro

Sometimes you'll want to do many things in a single tiddler. It's a good idea to put each command on a separate line, to make it easier to read. Unfortunately, this inserts blank lines in your tiddler when it's displayed — not very aesthetically pleasing.

This is where the <<silently>> macro comes in. Just like <<silently>>, it has a end macro — <<endsilently>>. Everything between the two is evaluated silently, without the user seeing anything.

There's one important trick to this macro: it doesn't show anything at all, including error messages. So if something isn't going right with your tiddler, try taking out the <<silently>> macro and see if an error message pops up.

This macro is ideal for setting up the state of the story at its outset. Here's how you might do things:

:: Startup
<<silently>><<tiddler '(setup)'>><<endsilently>>It's the first day of the school year...


:: (setup)
<<set classesCut = 0>>
<<set lunchMoney = 2.25>>

So, you've made it to the end! Those are all the macros built into Jonah. You should be able to do some fairly sophsticated things now. Give it a try!
