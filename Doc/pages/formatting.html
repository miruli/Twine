For example, in that newspaper passage we wrote in the last section, it sure would be nice to be able to italicize the name of the London Times. We can do that by surrounding the text with //s, like so:

:: the newspaper
If you believe what you read, the world is quiet. //The London Times'// front page is awash in some lurid tale of a fellow named John Lee whose execution failed thrice, and thus proven innocent -- or at least innocent enough to serve a life sentence.

Likewise, you can make text bold by putting double apostrophes around it, ''like this'', and you can underline text by putting two underscores around it, __like this__. There's a full list of formatting options here.

You can create lists by putting either a * or # at the beginning of a line. One use of this is to list actions a reader can take at the end of a passage, like so:

:: Start
It's a dark and stormy night aboard the [[Orient Express|http://en.wikipedia.org/wiki/Orient_Express]]. You can't sleep; something about the motion of the train disturbed you subtly. So instead you have elected to spend the night in the dining car, sipping coffee and perusing [[the newspaper]].

"Hello?" a female voice asks from behind you, perhaps from the entrance of the car.
* [[Turn around]]
* [[Keep reading]]

Notice that there isn't an empty line between the start of the list and the paragraph before it. That's a little idiosyncrasy of the way things are displayed. If you compile this new passage, you'll see that there's exactly the right amount of space.

Changing the font or other advanced tricks requires that you know at least a little CSS (a technology called Cascading Style Sheets), which is not something you pick up in a few minutes. Still, if you want to learn, this might be a good place to start. You can skip the parts that talk about HTML; you only need to know about the different CSS rules you can apply.

If you already know enough CSS to get into trouble, the basic pattern is:

@@CSS rule;another rule; text@@

So you could set some text in green Courier New by writing this:

@@font-family: "Courier New";color: green;This is not really the kind of thing that people would find attractive.@@

It's important not to put spaces between the semicolons.
