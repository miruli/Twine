To install Twine on Windows, run the installer you downloaded. This copies files to your Program Files folder and sets up an entry in your Start menu. On Macintosh computers, extract the contents of the zip file you downloaded to your Applications folder and then run the application inside. You'll see an interface like this:

To uninstall Twine on Windows, open the Add/Remove Programs control panel in Windows XP or earlier, or the Programs and Features control panel in Vista. Choose Twine from the list of applications and click the Uninstall button. On the Macintosh, all you need to do is delete the Twine folder from your Applications folder.
